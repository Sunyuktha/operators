Types of Operators

Arithmetic Operator

Arithmetic operators are used to perform numerical operations like addition, subtraction, multiplication, and division

Relational Operator

Relational Operators compare two entities to define or test a relationship between them.

Logical Operator

Logical Operators are symbols or words that connect expressions to perform logical operations and return a true or false value

Assignment Operator

Assignment operators assign values to variables in a program. The left side of the assignment operator is the variable, and the right side is the value
